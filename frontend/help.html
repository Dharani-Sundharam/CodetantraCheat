Installation & Setup

To install CodeTantra Automation:

  1. Download the zip file from our website
  2. Extract it to any folder on your computer
  3. Double-click on install.bat to run the installer
  4. Wait for the installation to complete
  5. Run run.bat to start the application

The installer will automatically check for Python, install required packages, and set up browser drivers.

CodeTantra Automation requires:

  • Windows 10 or Windows 11
  • Python 3.8 or higher
  • Internet connection for API calls and browser installation
  • Administrator rights for browser installation
  • At least 2GB of free disk space

If you see a "Python not found" error:

  1. Download Python 3.8+ from python.org
  2. During installation, make sure to check "Add Python to PATH"
  3. Restart your computer after installation
  4. Run the installer again

If the problem persists, try running the installer as Administrator.

To configure your accounts:

  1. Navigate to the desktop-app folder
  2. Copy credentials_template.py to credentials.py
  3. Open credentials.py in a text editor
  4. Fill in your CodeTantra account details
  5. Save the file and restart the application

Make sure to use the correct login URL for your institution's CodeTantra platform.

Using the Application

This is the most critical step for successful automation:

  1. Open your answers account in a web browser
  2. Navigate to the specific question you want to solve
  3. Open your target account in another browser tab
  4. Navigate to the EXACT SAME QUESTION in the target account
  5. Verify both accounts show the same question number and content
  6. Only then start the automation

The application will check if both accounts are synchronized before proceeding.

CodeTantra Automation supports:

  • Code completion problems (Type 1 and Type 2)
  • Multiple choice questions
  • Fill-in-the-blank questions
  • Various programming languages (Java, C++, C, Python, JavaScript)

The application automatically detects the question type and applies the appropriate solution method.

Our credit system works as follows:

  • Code completion questions: 5 credits each
  • Other question types: 3 credits each
  • Failed problems: 1 credit (still charged)
  • Free credits: 80 credits on signup
  • Referral bonus: 70 credits per successful referral

Credits are deducted in real-time as problems are processed.

Troubleshooting

If the application crashes:

  1. Check the error logs in the application window
  2. Close other applications to free up system resources
  3. Restart the application
  4. Ensure both CodeTantra accounts are still logged in
  5. Check your internet connection
  6. If the problem persists, contact support with the error details

If browser installation fails:

  1. Run the installer as Administrator
  2. Check your internet connection
  3. Disable antivirus temporarily during installation
  4. Try manual installation: python -m playwright install firefox
  5. Restart your computer and try again

If login fails:

  1. Verify your credentials in credentials.py
  2. Test login manually in your web browser
  3. Check if your accounts are not locked or suspended
  4. Ensure you're using the correct login URL
  5. Update credentials if they have changed

If typing issues occur:

  1. Close unnecessary applications to free up system resources
  2. Check if the CodeMirror editor is properly focused
  3. Ensure the browser window is not minimized
  4. Restart the application
  5. Check for any browser extensions that might interfere
