Phase 1: Installation from ZIP File
1

Download & Extract

Download the CodeTantra Automation zip file from our website and extract it to any folder on your computer.

Recommended Location: Extract to your Desktop for easy access.

After extraction, you should see these files:

  • desktop-app/ folder (contains all Python files)
  • install.bat (installation script)
  • run.bat (application launcher)
  • requirements.txt (dependencies list)
2

Run Installation

Double-click on install.bat to start the installation process.

Important: Make sure you have Python 3.8+ installed on your computer. If not, the installer will guide you to download it.

The installer will automatically:

  1. Check if Python is installed
  2. Install all required Python packages
  3. Download and install browser drivers (Firefox, Chromium)
  4. Set up the application for use
Success: Wait for the "Installation Complete" message before proceeding.
3

Verify Installation

After installation completes, verify that everything is working correctly:

  • Check that all files are in place
  • Ensure no error messages during installation
  • Verify Python packages are installed
  • Confirm browser drivers are available

The application is now ready for configuration.

Phase 2: Account Setup & Configuration
4

Create Web Account

First, create an account on our website to get your API credentials:

  1. Go to our website homepage
  2. Click "Register" and create an account with your educational email
  3. Verify your email address
  4. Note down your API credentials from the dashboard
Free Credits: You get 80 free credits on signup to get started!
5

Configure Desktop App Credentials

Set up your CodeTantra account credentials in the desktop application:

  1. Navigate to the desktop-app folder
  2. Copy credentials_template.py to credentials.py
  3. Open credentials.py in a text editor
  4. Fill in your CodeTantra account details
LOGIN_URL = "https://your-codetantra-url.com" ANSWERS_ACCOUNT = { "username": "your_answers_username", "password": "your_answers_password" } TARGET_ACCOUNT = { "username": "your_target_username", "password": "your_target_password" }
Security Note: Keep your credentials secure and never share them with others.
6

Configure API Settings

Set up API connection for credit tracking:

  1. Open desktop-app/config.py
  2. Update the API settings with your credentials
API_BASE_URL = "https://your-backend-url.com" API_KEY = "your_api_key_from_website" CREDIT_SYSTEM_ENABLED = True
7

Test the Setup

Verify everything is working correctly:

  1. Run run.bat to start the application
  2. Accept the disclaimer dialog
  3. Check if API connection is successful
  4. Verify your account credentials are working
Ready: If all checks pass, you're ready to proceed to the next phase.
Phase 3: Critical Preparation - Sync Both Accounts
CRITICAL WARNING

This step is ESSENTIAL for successful automation. The application will check if both accounts are on the same question. If they don't match, it will wait until you manually sync them.

8

Prepare Both CodeTantra Accounts

Before starting automation, you must ensure both accounts are on the EXACT SAME QUESTION:

  1. Open your answers account in a web browser (Chrome/Firefox)
  2. Navigate to the specific question you want to solve
  3. Open your target account in another browser tab/window
  4. Navigate to the EXACT SAME QUESTION in the target account
  5. Verify both accounts are on the same problem before starting
  6. Check question numbers match exactly
  7. Ensure both are on the same page/step
Double-Check: Take a moment to verify both accounts show identical question content and numbers.
9

Account Synchronization Verification

To ensure proper synchronization:

  • Both accounts should show the same question number
  • Question content should be identical
  • Both should be on the same step/page
  • No differences in question text or options
Pro Tip: If you're unsure, refresh both pages and navigate to the same question again.
Phase 4: Running the Application
10

Start the Application

Double-click run.bat to start CodeTantra Automation.

First Time: You'll see a disclaimer dialog. Read it carefully and click "I AGREE" to continue.

The application will:

  • Check API connection
  • Verify your credentials
  • Initialize the automation system
11

Configure Automation Settings

Set up your automation parameters:

  • Number of Problems: Set how many questions to solve (1-50)
  • Automation Mode: Choose between single run or endless mode
  • Question Types: Select which types to handle
Recommendation: Start with 1-2 problems to test the system before running larger batches.
12

Start Automation Process

Click "Start Automation" and watch the process unfold:

  1. Question Sync Check: Verify both accounts are on same question
  2. Question Detection: Identify the question type automatically
  3. Solution Extraction: Get answer from your answers account
  4. Code Transfer: Type solution in your target account
  5. Submission: Submit answer automatically
  6. Verification: Confirm successful submission
  7. Next Question: Move to next problem (if applicable)

Troubleshooting Common Issues

Installation Problems

Python Not Found: Download Python 3.8+ from python.org and ensure "Add Python to PATH" is checked during installation.

Package Installation Failed: Run Command Prompt as Administrator and execute: pip install -r requirements.txt

Browser Installation Failed: Run as Administrator, check internet connection, or try manual installation: python -m playwright install firefox

Account Issues

Login Failed: Verify credentials in credentials.py, test login manually in browser, check if accounts are locked.

Questions Don't Match: Ensure both accounts are on the SAME question, refresh both pages, check question numbers match exactly.

API Connection Failed: Check internet connection, verify API URL in config.py, ensure API service is running.

Performance Issues

Slow Typing: Close unnecessary applications, check system resources, restart the application.

Application Crashes: Check error logs, update dependencies, restart computer, contact support with error details.

Browser Issues: Close other browser instances, restart application, check if antivirus is blocking the browser.
